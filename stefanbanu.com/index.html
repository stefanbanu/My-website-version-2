Project

This is a simple Login desktop application that connects to a mysql database. For this application i used the following software programs:

  • As ide (integrated development enviroment) i used Eclipse.
  • For the gui (graphical user interface) i used WindowBuilder, a plugin for Eclipse.
  • I used the following jdbc (java database connectivity) driver for MySQL --> Download - mysql-connector-java-5.0.8.zip
  • In order to manage the database table etc. i used XAMPP (Cross-Platform (X), Apache (A), MySQL (M), PHP (P) and Perl (P).
